# 🚀 DevScore

> AI-powered GitHub profile analyzer that compares developers against real-world engineering roles ⚡

---

## 🌟 About DevScore

DevScore analyzes GitHub profiles using AI and developer metrics to generate:

- 🧠 Developer archetypes
- 📊 GitHub insights
- 💻 Language & tech stack analysis
- 🎯 Dream role compatibility
- 🚀 Engineering strengths & weaknesses

Simply enter a GitHub username and get an intelligent developer breakdown in seconds.

# ⚡ Installation

## 1️⃣ Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/devscore.git
```

## 2️⃣ Navigate into the project

```bash
cd devscore
```

## 3️⃣ Install dependencies

```bash
npm install
```

## 4️⃣ Setup environment variables

Create a `.env` file:

```env
VITE_GITHUB_TOKEN=your_github_token
OPENAI_API_KEY=your_openai_key
```

## 5️⃣ Run the development server

```bash
npm run dev
```
